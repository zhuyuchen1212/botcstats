Player format: Name Role [TeamHint]

One player per line. Autocomplete suggests names and roles as you type.

Names: Names can be anything — just be consistent so stats track correctly. Use underscores for spaces.

Jane Imp First name only
Jane_Doe Imp Full name (underscore = space)
JaneD Imp Nickname — whatever your group uses

Roles & team changes:

John Snake_Charmer+Witch Multiple roles (use +)
Mike Chef Good->Evil Team change (use ->)

Put team-changers in their FINAL team's section.

Storyteller: Multiple storytellers? Use + (e.g., Jane+Mike).

Fabled & Lorics: Optional comma-separated fields below the script. These are game-level modifiers, not player roles.
